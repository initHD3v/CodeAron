**CodeAron** adalah AI Coding Assistant berbasis CLI yang berjalan secara lokal (offline) di macOS (Apple Silicon). Didesain khusus untuk pengembangan Flutter dengan pendekatan *Clean Architecture*.

## Fitur Utama
- **Offline & Private:** Berjalan sepenuhnya di perangkat user (M1/M2/M3) menggunakan MLX.
- **Flutter Expert:** Memahami Clean Architecture, BLoC, dan Provider.
- **Safety First:** Integrasi Git otomatis, rollback jika build gagal, dan validasi statis.
- **Multimodal:** Mampu menganalisis desain UI (JPG/PNG) menjadi kode Flutter.
- **Bahasa Indonesia:** Interaksi natural dalam Bahasa Indonesia.

## Arsitektur
Sistem ini dibangun menggunakan Python 3.11+, MLX (Apple Silicon), SQLite, dan Tree-sitter.
